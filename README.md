# Portfolio Vinicius Ramos

Sobre o Projeto

Este repositorio contem o codigo-fonte do portfolio online de Vinicius Ramos. O site foi criado para apresentar os projetos, habilidades e informacoes de contato de Vinicius de forma visual, organizada e responsiva.

Demonstracao

Voce pode acessar a versao publicada do projeto pelo link abaixo:

https://izackzin.github.io/Portfolio-Vinicius-Ramos/

Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

Funcionalidades

- Pagina inicial com apresentacao pessoal
- Secao de projetos desenvolvidos
- Informacoes de contato
- Design responsivo para diferentes dispositivos
- Navegacao simples e intuitiva

Pronto! O portfolio sera exibido localmente no seu navegador.
